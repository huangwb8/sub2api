## 我们收集的信息

为提供本服务，我们可能收集以下信息：

### 账户与身份信息

- 用户名、昵称、邮箱地址、手机号、登录凭据
- 第三方登录标识、头像、公开资料或验证状态
- 你提交给客服、工单、申诉或实名认证流程的身份信息

### 交易与支付信息

- 订单号、商品信息、支付金额、币种、支付状态、退款状态
- 支付渠道返回的交易流水、付款时间、账单摘要和风控结果
- 充值余额、订阅状态、优惠券、代金权益和账务变动记录

通常情况下，我们不会自行保存完整银行卡号、支付密码或第三方支付账户口令；该类信息通常由相应支付服务商根据其自身规则处理。

### 使用、计费与请求日志

- API Key、请求时间、请求路径、模型名称、调用结果、错误码
- 用量统计、消耗额度、账单明细、速率限制、并发限制触发记录
- 设备标识、浏览器信息、操作系统、语言、时区、访问时间、点击记录
- IP 地址、网络信息、代理信息、登录日志、风险标签与异常行为特征

### 你主动提交的内容

如果你通过本服务发送提示词、消息、图片、文件、音频、输出结果反馈或其他内容，这些内容可能包含个人信息、商业信息或其他敏感数据。为了完成你发起的请求，这些内容可能被转发给相应的上游 AI 服务商或基础设施服务商。

### Cookie 与本地存储

我们可能使用 Cookie、本地存储、会话存储或类似技术，用于维持登录状态、保存语言与主题偏好、防刷与风控、改善页面体验以及统计服务运行情况。

你可以在浏览器中管理 Cookie，但这可能影响登录、支付、控制台或其他功能的正常使用。

我们可能将收集的信息用于以下目的：

- 创建与管理账户，完成登录、鉴权、权限控制和身份验证

- 进行财务对账、争议处理、合规审计、反欺诈、风险控制和安全分析

在适用法律允许的范围内，我们也可能基于去标识化、聚合化或统计化数据进行运营分析，但不会在未经授权的情况下以可识别个人身份的方式对外公开你的个人信息。

## 我们如何共享、转让与披露信息

除法律法规另有要求或本政策另有说明外，我们不会向无关第三方出售你的个人信息。

在下列情形下，我们可能共享必要信息：

- 与上游 AI 服务商共享：用于完成你发起的模型请求、内容处理、鉴权与路由
- 与支付与清结算服务商共享：用于完成支付、退款、对账、风控和争议处理
- 与云服务、CDN、日志监控、风控、安全审计、客服或消息通知服务商共享：用于支撑本服务运行
- 根据法律法规、行政命令、司法要求或监管调查进行披露
- 在合并、收购、资产转让、重组或类似交易中，在必要范围内向交易相对方及其顾问披露

我们要求相关接收方在其业务必要范围内处理信息，并采取合理的保密与安全措施。但你理解，发送给上游 AI 服务商或第三方支付服务商的数据，还将同时受到该第三方自身条款与隐私政策的约束。

## 跨境传输

由于本服务可能调用位于不同国家或地区的上游 AI 服务商、云基础设施或支付渠道，你的信息可能被传输、存储或处理于你所在司法辖区之外。

如果你所在地区对个人信息跨境传输有额外要求，你应在使用本服务前确认自己有权提交相关数据，并根据需要取得适当授权或采取必要合规措施。

## 信息保存期限

我们会在实现本政策所述目的所必需的最短期限内保存你的信息，除非更长的保存期限是法律法规要求、解决争议所必需，或为维护系统安全、审计、对账、反欺诈与风控所合理需要。

不同类型数据的保存期限可能不同，例如：

- 账户基础资料：在账号存续期间保存，并在注销后保留合理的法定或争议处理期限
- 订单、支付、退款与账单记录：按财务、税务、审计与合规要求保留
- API 调用日志、风险日志与安全日志：按运维、安全和反滥用目的保留
- 工单、申诉、客服沟通记录：按服务支持与争议处理需要保留

超过保存期限后，我们会根据适用法律及技术可行性，对相关信息进行删除、匿名化处理或安全隔离。

在适用法律规定的范围内，你通常享有以下权利：

- 查询我们是否处理了你的个人信息，并访问相关信息

- 注销账户、撤回同意，或限制特定处理活动

如你希望行使前述权利，可通过本政策载明的联系方式与我们联系。为保障安全，我们可能要求你完成合理的身份验证。法律法规规定的例外情形除外。

如果发生可能影响你个人信息安全的事件，我们会根据适用法律和实际情况采取补救措施，并在必要时向你通知相关情况。
